### 需加入契約的額外條款

#### 1. 稅務條款（租金淨額聲明）
```
本租金每月新台幣 45,000 元整，係指甲方（房東）之實收淨額。本租賃所得依法應由乙方（醫師）辦理扣繳。甲方同意上述租金款項外，由乙方另行負擔並代為繳納依稅法規定計算之租賃所得稅款（10%）及二代健保補充保費（2.11%）。乙方每月支付予甲方之金額為新台幣 45,000 元，並另向稅務及健保機關辦理申報繳納扣繳稅款及補充保費。甲方應配合提供身分證影本、銀行帳戶資訊及相關必要文件，供乙方辦理稅務申報使用。
```

#### 2. 裝修免租期條款
```
自民國 115 年 X 月 X 日起至 115 年 X 月 X 日止，計 30 天為裝修免租期。期間免收租金，承租人僅需負擔使用期間之水電費及管理費。正式起租日為 115 年 X 月 X 日，租金支付亦自該日起算。
```

#### 3. 診所用途聲明條款
```
乙方承租甲方之房屋作為中醫診所使用，並將依照政府相關規定完成必要之房屋使用申請及變更程序，以符合一切診所使用之規定，甲方已清楚了解且無異議。
```

#### 4. 開業執照保障條款（爭取，若無法則放棄）
```
本契約之履行，以乙方取得該址之「中醫診所開業執照」為必要條件。若因房屋土地使用分區（商業區/住宅區）、路寬限制、防火避難規定或既有違建問題，致主管機關無法核發執照，乙方得提前通知甲方解除契約，甲方應於通知後 7 日內退還已繳納之押金及預付租金。
```

#### 5. 租金遞增條款（6 年總期）
```
本租賃契約總租期共計 6 年。租金調整方式約定如下：
- 第 1–2 年：每月 45,000 元
- 第 3–4 年：每月 47,000 元
- 第 5–6 年：每月 49,000 元
（以上皆為房東實拿淨額，稅務依本契約稅務條款辦理）
期滿後，乙方享有優先續租權。
押金隨租金調漲同步調整，於調漲生效日前補足差額。
（第 3 年起需補繳 2,000 × 2 個月 = 4,000 押金差額）
```

## 與房東溝通話術

**核心目標：** 讓房東覺得「租給我不只是租房，是幫助一個年輕人創業和顧家」

**重點說明：**
- 在南京東路欣悅中醫已工作滿五年，業績穩定，從副院長位置離開
- 離開原因：回家讓父親順利退休，開一間自己的小診所
- 這個診所不只是工作場所，是我長期的家和事業根基
- 寶寶剛出生 5 個月，這個階段創業壓力很大，一間診所從零開始至少 300–500 萬，半年才能開業
- 張先生的合約條件對我們幫助很大，真的很感謝
- 第一次創業，有不懂的地方請直接告訴我，就像稅務扣繳那件事，電話裡提醒我才知道
