# 衛氣

## 摘要

衛氣是水穀精微中的剽悍之氣，行於脈外，不受經脈約束，具有護衛肌表、溫煦臟腑、調節腠理三大功能，其運行節律與睡眠密切相關。衛氣虛弱時易出現感冒、自汗、怕冷等症狀，可用玉屏風散治療。

---

## 正文

### 定義與性質

衛氣源於水穀精微，是其中剽悍（即剽疾、勇悍）的部分。其特性為行於脈外，不受經脈約束，運行迅速而靈活，遍布全身體表與臟腑之間。

### 主要功能

1. **護衛肌表**：抵禦外邪（風、寒、暑、濕、燥、火）入侵，是人體的第一道防線。
2. **溫煦臟腑肌肉皮毛**：維持臟腑、肌肉、皮毛的正常溫度與功能。
3. **調節腠理開合**：控制汗液的排泄，調節體溫與水液代謝。

### 運行規律

- **白天**：行於陽分（體表、六腑），循行 25 周。
- **夜晚**：行於陰分（五臟），循行 25 周。
- 合計一晝夜共循行 50 周，與人體睡眠節律密切相關。衛氣入陰則人得以入睡，衛氣出陽則人醒寤。

### 衛氣虛的病理表現

| 症狀 | 機制 |
|------|------|
| 易感冒 | 護衛肌表之力不足，外邪易於入侵 |
| 自汗 | 腠理調節失常，汗液外洩不固 |
| 怕冷（惡風畏寒） | 溫煦功能減弱，體表失於溫養 |

### 治療方劑

**玉屏風散**（出自《丹溪心法》）

| 藥物 | 功效 |
|------|------|
| 黃耆 | 補益衛氣，固表止汗 |
| 白朮 | 健脾益氣，助化生衛氣之源 |
| 防風 | 疏風散邪，引藥達表 |

三藥合用，補中有疏，固表而不留邪，適用於衛氣虛弱、表虛不固之證。

## 相關概念

- 營氣（與衛氣相對，行於脈內，主營養）
- 宗氣（積於胸中，主呼吸與心脈）
- 元氣（腎中精氣所化，先天之氣）
- 腠理（皮膚與肌肉之間的空隙，衛氣調節的場所）
- 營衛失調（衛氣與營氣失於協調的病理狀態）
- 玉屏風散
