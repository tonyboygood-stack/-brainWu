# 衝脈第一條路線

## 基本資訊

| 項目 | 穴位 |
| ---- | ---- |
| **開穴** | SP-4（公孫） |
| **循經第一點** | CV-2（曲骨） |
| **地標穴** | CV-2（曲骨）、KI-11（橫骨）、ST-30（氣沖）、KI-16（肓俞）、KI-21（幽門） |

---

## 走法

- **開穴**：SP-4（公孫）啟動衝脈
- **起點**：始於下腹部
- **湧出**：從恥骨軸線的 **CV-2（曲骨）** 出發，經 **KI-11（橫骨）**，在 **ST-30（氣沖）** 處衝脈之氣湧出
- **橫行**：往橫向走至 **SP-12（沖門）**，再折回中線
- **上行**：沿腎經依序經過 KI-11 橫骨 → KI-12 大赫 → KI-13 氣穴 → KI-14 四滿 → KI-15 中注
- **地標**：**KI-16（肓俞）**（地標穴）
- **終點**：繼續過 KI-17 商曲 → KI-18 石關 → KI-19 陰都 → KI-20 通谷 → KI-21 幽門（均距中線 0.5 寸），最後**瀰散入胸腔**

## 生理意義

- 引動腎的陽氣向上，幫助脾的陽氣
- 將多餘的**後天之氣**輸送至腎，以補充先天之氣

---

## 失衡病機與表現

**病機**：下焦阻滯停滯，中焦失去滋養並產生病理堆積

### 脈象
- 右脈 15 菽並延伸至中層
- 右關沉弱、右尺沉緩軟

### 舌象
- 隱樑（[[2026-04-25_衝脈的特殊舌診--隱樑]]）
- 舌中凹陷（中焦虛）

### 症狀

**下焦阻滯**
- 陰部疼痛、瘙癢、白帶
- 前列腺炎、尿失禁、小便不暢
- 痛經伴血塊、子宮肌瘤、子宮內膜異位症
- 大便帶黏液、血或痰、排便疼痛、腸漏症、便祕
- 水腫

**中焦虛夾濕（治則：健脾化濕）**
- 飯後腹脹
