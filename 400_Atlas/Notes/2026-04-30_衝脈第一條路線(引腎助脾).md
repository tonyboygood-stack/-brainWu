# 衝脈第一條路線（引腎助脾）

**別稱**：十二正經之海、後天之氣之海、五臟六腑之海

## 基本資訊

| 項目 | 內容 |
| ---- | ---- |
| **開穴** | SP-4（公孫）男左女右 |
| **循經第一點** | CV-2（曲骨） |
| **地標穴** | CV-2（曲骨）、KI-11（橫骨）、ST-30（氣沖）、KI-16（肓俞）、KI-21（幽門） |
| **古典主症** | 雞鳴泄（清晨腹瀉） |
| **一般脈象** | 腎陽無法感覺到在滋養脾臟 |

---

## 適應症

處理**後天之氣的虛證與實證**問題，包括：
- 分娩創傷、源於出生時脾氣虛的問題
- 雞鳴泄、水樣便、消化功能虛弱、食物消化不良
- 幼兒期開始的過敏
- 伴隨抑鬱的慢性疲勞
- 乳糜瀉（Celiac disease）
- 自出生以來生理和情感需求未得到滿足所帶來的後果

---

**完整路徑**：下腹部 → CV-2 → KI-11 → ST-30 → SP-12 → KI-11 → KI-12 → KI-13 → KI-14 → KI-15 → KI-16 → KI-17 → KI-18 → KI-19 → KI-20 → KI-21（瀰散入胸腔）

## 生理意義

- 引動腎的陽氣向上，幫助脾的陽氣
- 將多餘的**後天之氣**輸送至腎，以補充先天之氣

---

## 失衡病機與表現

**病機**：下焦阻滯停滯，中焦脾虛失去滋養並產生病理堆積

### 脈象
- 右脈 15 菽並延伸至中層
- 右關沉弱、右尺沉緩軟

### 舌象
- 隱樑（[[2026-04-25_衝脈的特殊舌診--隱樑]]）
- 舌中凹陷（中焦虛）

### 症狀

**下焦阻滯（氣滯血瘀痰濕熱瘀）**
- 陰部疼痛、瘙癢、白帶
- 前列腺炎、尿失禁、小便不暢
- 痛經伴血塊、子宮肌瘤、子宮內膜異位症
- 大便帶黏液、血或痰、排便疼痛、腸漏症、便祕
- 水腫

**中焦虛夾濕（治則：健脾化濕）**
- 濕痹
- 飯後腹脹、疲勞、食積、消化不良、食慾不振
- 脾虛不攝血：血虛血瘀出血
- 肝侵犯脾：壓力引起飲食失調（納差或暴飲暴食）

> 臨床案例：芮 xx，自律神經失調、睡前暴飲暴食、腹脹痛，用公孫＋內關有效，仍可進一步優化。

---

## 臨床處理（針刺流程）

1. **公孫**開穴，男左女右
2. 標定沖脈第一條 → 選擇地標穴：例如 [[氣衝(ST-30)]]、[[肓俞(KI-16)]]、[[幽門（KI-21）]]
3. 選定剩餘循經穴位，**從遠端向近端**進針
4. **重新巡查**每個穴位**非常細微的振動**
5. 確認脈象已**更深地移動到手腕深處**；若無，**重新震顫針**確保與**元氣層面**建立連接（透過**意念**而非實際深度）
6. （可選）給患者一句**肯定句**，留針期間大聲或默念，例如：「我的自然能量已恢復，我平靜且自由。」
7. 從第一個循經穴位插入時算起，**留針 40 分鐘**
8. 以**八脈奇經**手法切脈，記錄變化並告知患者
9. **拔針順序**：先拔**開穴**（公孫），其餘按**插入相反的順序**拔出
10. 請患者在治療台上靜躺**五分鐘**，最好獨處，供反思
