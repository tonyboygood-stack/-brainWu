# 病案：系統性肥大細胞增生（複雜案例）

> 值得深入研究病理機制與用藥思路。

## 西醫診斷
- 系統性肥大細胞增生（Systemic Mastocytosis）
- 肝硬化初期
- 肺高壓（近期超音波發現）

## 主訴與症狀
| 系統 | 症狀 |
|------|------|
| 血液 | 貧血 Hb: 8.0 |
| 心血管 | 心臟肥大（瓣膜正常）、血壓不穩、舒張壓偶降至 40-50、偶發休克、心悸 |
| 呼吸 | 肺高壓、乾咳、躺咳、無法平躺 |
| 水液 | 腳嚴重水腫 |
| 神經 | 頭暈、失眠 |
| 其他 | 女乳症、眼睛乾 |

## 脈象
- 脈緩大
- 左關鬱

## 辨證思路
從**肝**切入：
- 左關鬱 → 肝氣鬱滯
- 肝主疏泄失職 → 氣滯血瘀、水液代謝失常 → 水腫、肺高壓
- 女乳症、眼睛乾 → 肝血不足、肝氣橫逆

治法：**疏肝清肝、活血化痰**

## 治療進展

| 日期 | 脈象 | 症狀變化 |
|------|------|---------|
| 初診前 | 緩大，左關鬱，濕重 | 失眠、無法平躺、嚴重水腫、偶發休克 |
| 2026-04-30 | 濕氣減輕，虛損現 | 可睡、可平躺、水腫減少，續療 |

## 下一步觀察重點
- 虛損顯現後，補虛的時機與方向（氣虛？血虛？陰虛？）
- 肺高壓與中醫肺腎關係的對應
- 肥大細胞增生的中醫病機研究

## 學習方向
- 系統性肥大細胞增生的現代醫學病理
- 肝硬化初期的中醫辨治
- 活血化痰對肺高壓的機制
