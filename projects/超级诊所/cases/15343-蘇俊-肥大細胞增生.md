## 四、西醫病理串聯

| 西醫診斷 | 核心機制 |
|---------|---------|
| 系統性肥大細胞增生 | 肥大細胞浸潤多器官（肝/骨髓），組織胺大量釋放 → 血管擴張 → 血壓不穩 → 偶發休克 |
| 肝硬化初期 | 肥大細胞浸潤肝臟 + 門脈高壓 → 凝血障礙 → 貧血加重 |
| 貧血 Hb 8.0 | 骨髓肥大細胞浸潤 + 肝硬化脾亢 + 慢性出血傾向三因疊加 |
| 心臟肥大 | 長期低血紅素 → 心臟代償性肥大（高輸出性心臟病） |
| 肺高壓 | 肝肺症候群（hepatopulmonary syndrome）或肥大細胞浸潤肺血管 |
| 無法平躺、乾咳躺咳 | 肺靜脈壓升高 → 端坐呼吸（orthopnea） |
| 女乳症 | 肝硬化 → 肝無法代謝雌激素 → 男性乳房發育 |
| 舒張壓驟降 | 組織胺大量釋放 → 全身血管擴張 → distributive shock |

## 五、中醫病機分析

### 核心病機流程

```
先天腎虛或後天損傷
        ↓
   肝腎不足 → 衝脈失養（血海空虛）
        ↓
   肝鬱氣滯 → 血瘀（左關鬱、舌下脈絡瘀紫）
        ↓
   瘀血阻肝 → 肝硬化（肝藏血失職）
        ↓
   氣血兩虛 → 衝脈失固 → 衝氣上逆
        ↓
   上：心悸、頭暈、偶發休克、失眠、不能平臥
   中：女乳（衝脈循胸 + 痰氣凝結）
   下：腳水腫（脾腎陽虛，水濕不運）
```

### 四個並存層次

| 層次 | 病機 | 對應症狀 |
|-----|------|---------|
| **本虛** | 氣血兩虛、肝腎不足 | 貧血、脈虛大、舌淡 |
| **血瘀** | 肝鬱血瘀、衝脈瘀阻 | 舌下脈絡怒張、肝硬化、左關鬱 |
| **衝氣上逆** | 衝脈失固，逆氣上衝 | 心悸、頭暈、偶發休克、不能平臥、乾咳躺咳 |
| **水飲** | 脾腎陽虛，水泛 | 腳水腫、肺高壓、眼乾（津液不布） |

## 六、衝脈病機解析

> 《靈樞》：「衝脈者，五臟六腑之海……其上者，出於頏顙……其下者，注少陰之大絡」  
> 《難經》：「衝脈為病，逆氣而裡急」

本案「偶發休克頭暈心悸」「無法平躺」「乾咳躺咳」，正是**衝氣上逆**的經典表現，根本是**血海空虛、衝脈失固**。

**西醫–中醫對應**：  
肥大細胞釋放組織胺 → 血管驟張 → 血壓崩塌  
＝ 氣不攝血、衝脈不固、陽氣外散

## 七、治療思路

> **首要原則：扶正為主，不可峻攻**  
> Hb 8.0 + 肺高壓 + 偶發休克 → 大幅行氣破血有風險

| 治法 | 方向 | 注意事項 |
|-----|------|---------|
| 建中固衝 | 小建中湯、黃耆建中湯方向 | 先穩住衝脈根基 |
| 柔肝活血 | 當歸芍藥散、一貫煎方向 | 不可用丹參大劑破血 |
| 溫陽利水 | 真武湯方向（水腫、肺高壓） | 附子劑量謹慎，注意血壓 |
| 降衝鎮逆 | 桂枝茯苓丸 + 牛膝引血下行 | 搭配補虛才安全 |

**女乳症附注**：肝鬱 + 痰氣凝結於乳絡，可考慮橘核、荔枝核、海藻類入方。

## 八、學習重點整理

1. **西醫「肥大細胞組織胺釋放」= 中醫「伏邪外發、氣不攝津血」**——兩套語言描述同一現象
2. **衝脈病不只是婦科**：凡逆氣上衝 + 血虛 + 不能平臥，皆要考慮衝脈
3. **舌下脈絡怒張**是肝硬化的中醫早期警示，診斷靈敏度高於舌面
4. **脈「緩大」= 虛大脈**，非實熱洪大，判斷須結合全局
5. 複雜病案的中醫思路：先找**主軸病機**（本案為衝脈失固），再分層處理
