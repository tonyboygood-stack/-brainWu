|------|------|------|

| 代扣租賃所得稅（10%） | 45,000 × 10% | $5,000 |
| 代扣二代健保補充保費（2.11%） | 45,000 × 2.11% | $1,055 |
| **我方每月總支出** | | **$51,055** |

## 二、額外增加的合約條款

### 條款 A：租金稅務扣繳說明
```
本租金每月新台幣 45,000 元整，係指甲方（房東）之實收淨額。
本租賃所得依法應由乙方（醫師）辦理扣繳。甲方同意上述租金
款項外，由乙方另行負擔並代為繳納依稅法規定計算之租賃所得
稅款（10%）及二代健保補充保費（2.11%）。乙方每月支付予甲
方之金額為新台幣 45,000 元，並另向稅務及健保機關辦理申報
繳納扣繳稅款及補充保費。甲方應配合提供身分證影本、銀行帳
戶資訊及相關必要文件，供乙方辦理稅務申報使用。
```

### 條款 B：裝修免租期與正式起租
```
自民國 115 年 X 月 X 日起至 115 年 X 月 X 日止，計 30 天為
裝修免租期。期間免收租金，承租人僅需負擔使用期間之水電費
及管理費。正式起租日為 115 年 X 月 X 日，租金支付亦自該日
起算。
```

### 條款 C：診所使用用途聲明
```
乙方承租甲方之房屋作為中醫診所使用，並將依照政府相關規定
完成必要之房屋使用申請及變更程序，以符合一切診所使用之規
定，甲方已清楚了解且無異議。
```

### 條款 D：開業執照保障條款（重要，嘗試爭取）
```
本契約之履行，以乙方取得該址之「中醫診所開業執照」為必要
條件。若因房屋土地使用分區（商業區/住宅區）、路寬限制、防
火避難規定或既有違建問題，致主管機關無法核發執照，乙方得
提前通知甲方解除契約，甲方應於通知後 7 日內退還已繳納之
押金及預付租金。
```

### 條款 E：租金遞增與押金調整
```
本租賃契約總租期共計 6 年。租金調整方式約定如下：
  第 1–2 年（115年X月X日 至 117年X月X日）：每月 45,000 元
  第 3–4 年（117年X月X日 至 119年X月X日）：每月 47,000 元
  第 5–6 年（119年X月X日 至 121年X月X日）：每月 49,000 元

上述金額皆以「房東實拿淨額」計算，稅務扣繳依本契約稅務條
款辦理。期滿後，乙方享有優先續租權。

押金應隨租金調漲同步調整，於調漲生效日前補足差額。
（即第3年起補繳 $2,000 × 2個月 = $4,000 押金差額）
```

## 四、額外條款的附件處理方式

### 標準做法：印出來作為附件釘在後面

如果額外條款寫不進合約本子，按以下步驟處理：

**Step 1：在主約內文加入引用句**
在主約空白處或備註欄手寫一行：
> 「雙方另有補充條款詳見附件一，與本契約具同等法律效力。」

**Step 2：附件事先列印好帶去**
- 不要現場手寫，事先印好、格式清楚
- 若現場需要修改某條，手寫修改後，**雙方在修改處旁邊簽名確認**

**Step 3：每頁都要雙方蓋章或簽名**
- 附件每一頁，甲乙雙方各簽名或蓋章一次
- 不能只在最後一頁簽

**Step 4：騎縫章（防止偷換頁面）**
- 將主約＋附件全部疊好釘起來
- 印章**跨頁縫蓋**，讓印文一半在這頁、一半在下一頁
- 每個頁縫都要蓋到
- 目的：日後若有人偷換中間頁面，騎縫章對不起來即可發現

**Step 5：正本兩份，每份都要有附件**
- 張先生一份、你一份
- 兩份都必須附有附件，且都要有騎縫章

### 條款優先順序建議
| 條款 | 難度 | 說明 |
|------|------|------|
| A（稅務扣繳） | 低 | 對張先生有利，應不會拒絕 |
| C（診所用途） | 低 | 說明用途，保護雙方 |
| B（免租期） | 中 | 已口頭談好，書面確認即可 |
| E（六年遞增） | 中 | 給張先生長期穩定感，通常接受 |
| D（執照保障） | 高 | 說明這是保護雙方、非找麻煩，耐心解釋 |
