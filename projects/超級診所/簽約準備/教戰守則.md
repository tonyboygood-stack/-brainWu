# 簽約教戰守則
> 目標：2026-04-27（日）與房東完成租約簽署

---

## 一、簽約前確認清單

### 房屋基本資料核查
- [ ] 確認房東為真正產權所有人（查土地登記謄本）
- [ ] 確認房屋無查封、假扣押、抵押等限制
- [ ] 確認房屋用途（地目/使用分區）**可合法作為診所**
- [ ] 確認消防、建管設施符合醫療場所要求

### 合約條款重點確認
- [ ] **租期**：起迄日、續約條件
- [ ] **租金**：金額、調漲條款、繳付方式
- [ ] **押金**：金額（通常 1–3 個月）、退還條件
- [ ] **裝潢權利**：是否可裝潢、裝潢歸屬（離開時是否需恢復原狀）
- [ ] **提前解約**：違約金計算方式
- [ ] **轉讓/轉租**：是否允許（若未來轉讓診所需注意）
- [ ] **使用目的**：明確載明「作為中醫診所使用」
- [ ] **稅費負擔**：房屋稅、地價稅由誰負擔

### 法規相容性
- [ ] 確認地址可申請衛生局診所設立許可
- [ ] 確認可申請健保特約
- [ ] 確認符合醫療法相關設置標準

---

## 二、談判重點與策略

### 我方核心需求
1. 租期至少 3 年（診所設立成本高，需攤提）
2. 裝潢自由度高，且離開時不需全數恢復原狀
3. 使用目的明確載入合約

### 房東可能在意的點
- 租金穩定性 → 強調診所為長期穩定用途
- 房屋維護 → 承諾妥善使用，定期維護
- 鄰里關係 → 說明診所客群特性（不吵雜）

### 讓步空間
- 租金可小幅調整，但裝潢條款不讓步
- 押金可多付，換取較長租期保障

## 三、簽約當天流程

1. **攜帶文件**
   - 身分證、健保卡
   - 印章
   - 頤固顧問聯絡窗口（備詢）
   - 預備金（押金現金或轉帳）

2. **簽約前再次確認**
   - 逐條朗讀合約，遇不清楚立即提問
   - 不急於當場簽，可要求帶回審閱

3. **簽約後**
   - 取得正本（至少一份）
   - 拍照留存每一頁
   - 確認押金收據
